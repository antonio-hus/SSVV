# GymTrackerPro Testing Documentation

## Overview

A comprehensive overview of the Task 3 testing strategy is provided in `GymTrackerPro_Task3_Summary.pdf`.

The GymTrackerPro frontend (`/gymtrackerpro/app/` and `/gymtrackerpro/components/`) is tested using Frontend Testings (FT) techniques.

All tests are implemented using **Jest**, **React Test Library**, and **Istanbul** for code coverage reporting.
The final test execution achieved:

* **100% Statements Coverage**
* **100% Branch Coverage**
* **100% Function Coverage**
* **100% Line Coverage**
